# PaperAssassin

적을 피해 버튼을 눌러 문을 열고 목표 스크롤에 도착하는 게임

조작키
- 움직임: W A S D
- 대쉬: Shift 

모든 기능들은 블루프린로 개발되었습니다.

기능들
- 버튼을 누르면 문이 열림, 일정시간이 지나면 닫힘 
- 지정구역을 순찰하다 플레이어를 탐지하여 공격하는 NPC
- 코인에 서 먹을 수 있음
- 일정 대쉬 속도 이상이 되면 먹은 코인을 드랍
- 드랍된 코인은 다시 먹을 수 있음
